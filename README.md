# Retail Store Sales Dashboard - Power BI

## Project Overview
Interactive Executive Summary Dashboard built in Power BI
analyzing retail store sales data across India.

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (M Language)

## Dataset
- 4 Tables: Customers, Products, Sales, Stores
- Time Period: 2022 - 2026

## Key KPIs
- Total Sales Amount: 1.05bn
- Total Net Sales: 882.09M
- Total Profit: 174.46M
- Total Quantity Sold: 55K
- Total Orders: 10K
- Profit Margin: 19.78%

## Dashboard Features
- Monthly Sales Trend (Line Chart)
- Region wise Sales (Bar Chart)
- Category wise Profit (Bar Chart)
- Store Type breakdown (Donut Chart)
- Product wise Sales (Bar Chart)
- Payment Mode Analysis (Bar Chart)
- 4 Interactive Slicers (Year, Month, Region, Payment Mode)
